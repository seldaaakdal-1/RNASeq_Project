Query RNA-Seq Data

GDCquery() is used to query RNA-seq data from the GDC portal.

query_TCGA = GDCquery(
  project = "TCGA-LIHC",
  data.category = "Transcriptome Profiling", # parameter enforced by GDCquery
  data.type = "Gene Expression Quantification",
  workflow.type = "STAR - Counts")
